# רצף (Retzef) — Spec

זיכרון חיצוני שקט למי שקופץ הרבה בין דברים. מחזיר אותך לרצף אחרי מעברים והפרעות,
ונותן קריאת-קשב יומית — הכול מקומי, בלי צילומי מסך.

---

## עקרונות מנחים

1. **Pull-only ושקט.** בלי pop-ups, בלי הפרעות. אתה מציץ מתי שבא לך.
2. **Ground truth.** לכידה אוטומטית של מה שקורה בפועל — לא דיווח עצמי.
3. **קשב, לא חלונות.** עוקב אחרי מה שנתת בו קשב (focus + זמן שהייה), לא אחרי מה שפתוח.
4. **מקומי לגמרי.** הכול נשמר על ה-Mac. אפס ענן, אפס וידאו, שורת טקסט אחת לכל צעד.

---

## שלב 1 — MVP

### פיצ'רים
1. **לכידה פסיבית** — כל ~4 שניות: אפליקציה פעילה + כותרת חלון + URL (בדפדפן), דרך `osascript`.
2. **מיון לחוטים:**
   - מסנן זמן שהייה (הצצות קצרות לא נספרות).
   - איחוד לפי פרויקט (מילות מפתח בכותרת/URL → העורך + הטאבים שלו = חוט אחד).
   - תקשורת (סלאק/וואטסאפ/מייל/צ'אט) → מגירה אחת.
   - פגישות (Zoom/Meet) → ציון-זמן, וגם גבול שסוגר סשן.
3. **לוח menu-bar** — 3–6 חוטים, לכל אחד השורה האחרונה (קובץ/טאב) + מתי.
4. **"פתח"** — פותח מחדש את הטאב/קובץ האחרון של החוט.
5. **אחסון מקומי** — JSON ב-userData, ניקוי אוטומטי של מה שמעל 3 ימים.

### לא ב-MVP (backlog)
- pop-ups / hotkey חובה.
- שכבת AI לניסוח "הצעד הבא".
- שחזור **כל** צביר החלונות (MVP פותח רק את האחרון).
- קלף בוקר ב-The Bridge.
- בלם ידני (merge/split חוטים).

## שלב 2 — קלף קשב (על אותו data, בלי לכידה חדשה)

- **מדדים:** מעברי-הקשר (בין חוטים — *לא* בין חלונות), הבלוק הרצוף הארוך ביותר, מספר החוטים.
- **גרף מיקוד לפי שעה** — מתי היית עמוק, מתי מפוזר.
- **המלצה אחת אדיבה** + חיבור לפעולה בכלים הקיימים (Worktime Blocker / Focus Timer).
- שני תנאים מחייבים:
  1. לספור מעברי-**הקשר**, לא מעברי-חלון.
  2. נדנוד אחד ואדיב — לא לוח-בושה.

---

## טכנולוגיה

- **Electron 31** (עקבי עם Focus Timer / News Ticker).
- **לכידה:** `osascript` (AppleScript) — אפס תלות נייטיב.
- **UI:** Tray + חלון פאנל (HTML/CSS/JS), RTL, light/dark.
- **אחסון:** JSON ב-`app.getPath('userData')`.

### הרשאות (נקודת החיכוך היחידה)
- **Accessibility** — כדי לקרוא כותרת חלון.
- **Automation** (פר-דפדפן) — כדי לקרוא URL של הטאב.
אישור חד-פעמי בפעם הראשונה שמריצים את Retzef.app.

## סיכונים ידועים

- קריאת URL דורשת הרשאת Automation לכל דפדפן בנפרד.
- מיון היוריסטי לא מושלם (טאב עמום, פרויקטים עם שם דומה) → בלם ידני נדחה ל-backlog.
